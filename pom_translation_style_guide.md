## 一、統一術語表

以下術語在四個客人之間有不同翻法。**粗體**為本系統統一用法，依據：四家出現頻率最高 + 聚陽內部慣用。

### 身體部位 / 結構

| 英文 | ✅ 統一用法 | ❌ 不用 | 說明 |
|------|------------|---------|------|
| Armhole | **袖襱** | 袖窿 | 「袖襱」四家共 548 次 vs「袖窿」36 次。窿是舊式用法 |
| Rise / Crotch | **襠** | 檔、浪 | 前襠、後襠、襠縫、襠寬。「檔」是早期誤用，「浪」是口語 |
| Gusset | **襠片**（褲襠位）/ **插片**（其他位置） | 嵌片 | 褲類用「襠片」；腋下、側邊用「插片」 |
| Yoke | **約克** | 育克 | 四家統一用「約克」（157 次），無人用「育克」 |
| Raglan | **拉克蘭** | 插肩 | 拉克蘭 190 次 vs 插肩 3 次 |
| Dart | **褶** | 省 | 「褶」339 次。「省」在成衣業常見但這批資料不用 |
| Princess Seam | **公主線** | 公主縫 | ATH/BNN/ONY 用「公主線」；GAP 用「公主縫」 |
| Cuff | **克夫** | 袖口羅紋 | 「克夫」254 次，指獨立的袖口構件。「袖口」指開口本身 |
| Hem | **下襬** | 底邊、襬邊 | 「下襬」468 次最多。「底邊」用於 band 底緣 |
| Half Moon | **半月貼** | 半月 | GAP 用「半月貼」較精確，指貼片不是形狀 |

### 構造 / 做工

| 英文 | ✅ 統一用法 | ❌ 不用 | 說明 |
|------|------------|---------|------|
| Snap | **暗釦** | 按釦、暗扣 | 「暗釦」117 次 vs「按釦」32 次 |
| Placket | **門襟**（前開）/ **袖衩**（袖口開叉） | 開叉 | 前開衫用「門襟」；袖口用「袖衩」 |
| Welt Pocket | **袋唇口袋** | 嵌線口袋 | 「袋唇」193 次，四家一致 |
| Cargo Pocket | **側邊口袋** | 工作口袋、貨袋 | ONY+GAP 主流用法 |
| Binding | **滾條** | 包邊 | 「滾條」37 次。「包邊」26 次但語義較廣 |
| Trim | **剪接**（結構性）/ **裝飾**（非結構） | — | 「剪接」330 次，指拼接片。裝飾帶、裝飾條用「裝飾」 |
| Drawcord / Drawstring | **抽繩** | 拉繩 | 「抽繩」58 次 vs「拉繩」5 次 |
| Vent / Slit | **開衩** | 衩 | 指下襬或側邊的開口 |

### 量法動詞

| 英文 | ✅ 統一用法 | ❌ 不用 | 說明 |
|------|------------|---------|------|
| Along Curve | **沿曲線** | 沿弧線、沿彎量、沿弧度 | 「沿曲線」118 次最多，四家都用 |
| Relaxed | **鬆量** | 放鬆、鬆弛 | 「鬆量」427 次，壓倒性 |
| Extended / Stretched | **拉量** | 拉伸 | 「拉量」264 次 vs「拉伸」54 次 |
| 2 Point Measurement | **兩點量** | 2 點測量、2 點側量 | 簡潔統一 |
| Tape Flat | **量尺平放量** | 皮尺平放測量 | BNN 用法最簡潔 |
| Tape Standing | **量尺站立** | — | 對應 Tape Flat |

## 二、翻譯格式規則

### 結構公式

POM 中文名的標準結構：`部位名 + 量法/位置修飾`

| 英文結構 | 中文結構 | 範例 |
|----------|----------|------|
| {POM} at {Position} | {POM}於{位置} | Collar Height at Center Back → 領高於後中 |
| {POM} from {Reference} | {POM}距{參考點} | Hip Position from___ → 臀位距___ |
| {POM} - {Method} | {POM} - {量法} | Armhole Straight - 2 Point → 袖襱直量 - 兩點量 |
| {Placement} from {Edge} | {位置}距{邊} | Pocket Placement from Center Front → 口袋位置距前中 |
| Front/Back {POM} | 前/後{POM} | Front Rise → 前襠 |

### 介詞用法

| 英文 | 中文 | 用法 |
|------|------|------|
| at | 於 | 位置描述：「領高**於**後中」「口袋高**於**中間」 |
| from | 距 | 距離起點：「臀位**距**腰」「鈕釦位置**距**領縫線」 |
| from (起始) | 從 | 量測起點：「袖長**從**後中」「**從**側縫到肩帶底部」 |
| to | 到/至 | 量測終點：「前中**到**後中」「邊**至**邊」 |
| along | 沿 | 量法：「**沿**曲線」「**沿**縫線」 |

**不用「在」**（「於」四家共 2,294 次 vs「在」363 次）。

### 縮寫與全稱

| 英文 | 中文 | 備註 |
|------|------|------|
| HPS / High Point Shoulder | 肩高點 | 四家統一，不縮寫 |
| CF / Center Front | 前中 | 不用 CF |
| CB / Center Back | 後中 | 不用 CB |
| SS / Sideseam | 側縫 | 不用 SS |

### 括號用法

- **量法補充**用括號：「袖襱直量（兩點量）」
- **位置補充**用括號：「領高（後中）」→ 但新版統一改成「領高於後中」
- **客人指定**用 (G)：「(G) 肩高點」「(G) 側縫到側縫」— 保留不翻

## 三、易混淆辨析

### 「袖口」vs「克夫」vs「羅紋」

- **袖口**（Sleeve Opening）：袖子末端的開口，量的是圍度
- **克夫**（Cuff）：縫在袖口的獨立構件（翻折式、鈕釦式等）
- **羅紋**（Rib）：針織彈性收邊，用在袖口、下襬、領口

「Sleeve Cuff Height」→「袖克夫高」（指克夫構件的高度，不是袖口開口）

### 「下襬」vs「底邊」

- **下襬**（Hem / Bottom Sweep）：整件衣服的底邊線 / 底邊圍度
- **底邊**（Bottom Edge）：構件的下緣（如口袋底邊、腰頭底邊）

### 「門襟」vs「袖衩」

- **門襟**（Front Placket）：前開衫的鈕釦開口
- **袖衩**（Sleeve Placket）：袖口的開叉（用於需要釦起的襯衫袖）

### 「腰圍」vs「腰頭」vs「腰帶」

- **腰圍**（Waist）：身體腰部的圍度
- **腰頭**（Waistband）：褲頭/裙頭的構件
- **腰帶**（Belt）：穿過褲袢的帶子

### 「襠」相關

- **前襠**（Front Rise）：前中腰頭到襠底的距離
- **後襠**（Back Rise）：後中腰頭到襠底的距離
- **襠寬**（Crotch Width）：襠底的寬度
- **襠片**（Gusset）：加在襠底的菱形或三角形補片
- **襠縫**（Rise Seam / Crotch Seam）：前後襠接合的縫線

### 「剪接」vs「裝飾」

- **剪接**（Trim，結構性）：拼接在衣身上的布片，如「剪接片高」「剪接寬」
- **裝飾**（Trim，非結構性）：緞帶、蕾絲、燙畫等附加物

## 四、客人特殊用法

### GAP 特點

- 偏好簡寫：「襱直2點量」（其他家寫「袖襱直量 - 兩點量」）
- 用「(G)」標記 GAP 專屬量法：「(G) 側縫到側縫」
- 「公主縫」（其他家用「公主線」）
- 「半月貼」比「半月」更精確
- 有時保留英文不翻：少數 Along Curve 條目直接寫英文

### ATH (Athleta) 特點

- 運動服術語較多：功能性肩帶、胸罩架、調節環等
- 「袖窿」出現 23 次（其他家很少），是 ATH 翻譯者的個人習慣
- 部分條目未翻譯直接保留英文（67 筆）

### BNN (Banana Republic) 特點

- 翻譯最完整（僅 1 筆未翻譯）
- 用字最統一，偏差最小
- 「袖衩」出現 5 次（門襟 / 袖衩 區分做得最好）

### ONY (Old Navy) 特點

- 資料量最大的客人（2,939 筆）
- 「滾條」用得最多（17 次 vs 其他家 3-10 次）
- 48 筆未翻譯保留英文（主要是非標準備註）

## 五、翻譯注意事項

1. **不要自己造詞**：如果不確定，先查四家翻譯表有沒有類似條目，學習既有風格再翻。
2. **保留英文的情況**：(G) 標記、POM Code（A1/B25/O4）、品牌名、「PLUS」「PETITE」「TALL」等 body type 名稱。
3. **數字和尺寸保留原文**：「1"」「2 1/2"」「1/4"」不翻成中文。
4. **量法說明不是 POM 名稱**：「* vendor grade」「11.30.22: RVSD TO MATCH ALL THE FIT SMPLS」這類是備註，不是 POM，不進 pom_dictionary。
5. **同一個 POM Code 在不同客人可能有不同英文名**：以 Centric 8 系統的英文名為正本，不要用客人自定義的變體名。

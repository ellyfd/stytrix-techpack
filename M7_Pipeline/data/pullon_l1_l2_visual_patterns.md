# PullOn 五階反推視覺特徵庫

> 從 m7_report 1180 PullOn × 五階字典反推
> 用途：VLM L1 偵測時，用具體 L2 視覺 pattern 為判斷依據
> 來源：bible 五階層展開項目 + L2 視覺差異分析報告 + m7_report 真實 step

---

## 判斷邏輯

VLM 不要直接判 L1，要先判**具體 L2 pattern**。**看到任一 L2 pattern → 列對應 L1**。
**所有 L2 pattern 都看不到 → 不列該 L1**。

## L1 = 腰頭 (WB) — PullOn 真實 L2 分布

| L2 | PullOn 占比 | 視覺 pattern（看到就算 WB）|
|---|---|---|
| 剪接腰頭_整圈 | **86%** | 腰部有明顯橫向剪接線分隔腰頭與大身，腰頭為獨立寬帶（含鬆緊抽繩變體）|
| 反折腰頭 | 11% | 大身布直接反折，沒剪接線，腰口有鬆緊聚攏皺褶 |
| 外露腰頭 | 3% | 運動褲腰部有外露鬆緊帶（直接可見彈性帶）|

> ⚠️ 沒看到上述任一 → 不列 WB

## L1 = 繩類 (DC) — PullOn 抽繩 ⚠️ 最容易誤判

| L2 | PullOn 占比 | 視覺 pattern（必須看到才能列 DC）|
|---|---|---|
| 腰繩 | **78%** | **看到腰部有繩頭從腰頭兩側穿出**，繩端有束口/打結/金屬頭 |
| 其它 | 18% | 其他位置抽繩 |
| 褲口繩 | 1% | 褲口位置有繩頭穿出（jogger）|

> ❌ **沒看到繩頭、繩孔、抽繩穿出** → 不要列 DC
> ❌ **只看到鬆緊聚攏皺褶（無繩頭）→ 是 PD 抽褶，不是 DC**
> 內綁帶/隧道繩等 sketch 看不見 → 不列

## L1 = 褶 (PD) — PullOn 真實褶子

| L2 | PullOn 占比 | 視覺 pattern |
|---|---|---|
| 死褶 | **34%** | 短直線凸起（dart），約 6 吋長，固定不動 |
| 抽褶_平車 | 28% | 整圈聚攏皺褶（**這是腰頭鬆緊紋路！沒繩頭就是 PD 不是 DC**）|
| 抽褶_其它 | 14% | 其他抽褶方式 |
| 活景褶 | 5% | 後片 V型活景褶 |

> ⚠️ PullOn 鬆緊腰頭聚攏 = PD（抽褶_平車/鬆緊帶）
> ⚠️ 不要把 PD 跟 DC 混淆（DC 必須看到繩頭）

## Sketch 通常看不見的 L1（PullOn 不要列）

- **LB 商標** → 多在後內側 / 腰頭內側
- **LI 裡布** → 內裡看不見
- **OT 其它** → 雜項做工（游線、對條格、整燙）
- **NT 領貼條** → PullOn 沒領
- **BN 貼合** → 無車線痕跡

## PullOn 不會出現的 L1（屬於上衣）

- AE / AH（袖孔/袖圍）/ SH（肩）/ SL（袖口）/ SP（袖叉）/ TH（拇指洞）
- NK（領）/ NP（領襟）/ HD（帽子）
- BM（上衣下襬，PullOn 用 LO 取代）
- SA（上身剪接，PullOn 用 SB 取代）
- ST（肩帶）/ DP（裝飾片）/ KH（Keyhole）

## 給 VLM 的判斷流程

```
For each L1 in [WB, PS, LO, PK, DC, PD, BS, SB, BP, FY, FP, LP]:
    1. Look at the L2 visual patterns listed for this L1
    2. Can I identify ANY specific L2 pattern in the sketch?
    3. If YES → list this L1 + name the matched L2
    4. If NO → do NOT list this L1
```

**關鍵 disambiguation**：
- DC 必須看到**繩頭**（cord ends）才列
- PD 看到聚攏皺褶但沒繩頭 → 列 PD（不要混 DC）
- SB 看到**非結構縫**的設計剪接 → 列 SB（不要漏）
- PullOn 看到前中 J 型 → FY_002 假前立（不是 PL）
